# Land Area Calculator

A responsive web application for converting land area measurements across multiple units commonly used in real estate and surveying.

## Features

- **10 Unit Support**: Convert between inch, feet, square feet, yard, square yard, gunta, cent, ankanam, acre, and hectare
- **Real-time Conversion**: Instantly see converted values as you type
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Beautiful UI**: Modern gradient design with smooth animations
- **Accurate Calculations**: Based on precise conversion factors

## Conversion Factors

The application uses the following exact measurements:

- 1 inch = 2.5 cm
- 1 foot = 12 inches or 30 cm
- 1 yard = 3 feet
- 1 sq yard = 9 sq feet
- 1 ankanam = 8 sq yard = 72 sq feet
- 1 cent = 435.6 sq ft
- 1 gunta = 2.5 cents
- 1 acre = 100 cents or 40 gunta
- 1 hectare = 2.47 acres

## How to Use

1. Open `index.html` in any modern web browser
2. Enter a value in the input field
3. Select the unit of measurement from the dropdown
4. View the converted values in all other units below
